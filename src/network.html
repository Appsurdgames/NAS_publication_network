NAS Publication Network

An interactive visualization of the publication network at the Numerical Analysis and Computational Mechanics (NAS) department at TU Delft.

About the Network

Each node represents a researcher. Nodes are connected if they have co-authored at least one paper together. The size of each node indicates the number of papers published by that researcher. The node colours:

  • Blue nodes: Current NAS employees
  • Red nodes: Former NAS employees
  • Gold nodes: Former NAS Master students
  • Green nodes: External researchers

How to Use

Explore the network: Click on any node to view researcher details, papers, and collaborators.

Search: Use the search box to find researchers by name or abbreviation.

Filters:

  • Min. shared papers: Show only connections with at least this many co-authored papers
  • Min. papers per author: Show only researchers with at least this many publications
  • NAS only: Display only current NAS employees
  • Show labels: Toggle researcher names on the network

Download: Use the "Download network" dropdown to export the network in JSON or CSV formats for external analysis.
